# Canvas App

A Canvas LMS client application built with Flutter.

## Getting Started
Clone the repository and run `flutter pub get` to install the dependencies.
then run the app with `flutter run`.

## Features
- [x] Chat with AI
- [x] Flashcards
- [x] Calendar
- [x] Assignments
- [x] Courses
- [x] Modules
- [x] Settings
- [ ] Quiz Submission
- [ ] Grades
- [ ] Notifications
- [ ] Announcements
